# Frontend AGENTS

- Follow root `AGENTS.md`.
- Frontend stack is Next.js App Router, TypeScript, Tailwind CSS, TanStack Query, Recharts, and SSE.
- The dashboard is an operations console, not a price-board clone.
- UI states must distinguish INITIALIZING, LIVE, DEGRADED, BACKFILLING, STALE, and ERROR.
- Do not add decorative UI that hides operational status or failure signals.
